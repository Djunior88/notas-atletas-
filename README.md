# Notas Atletas 🏅

## Sobre o projeto

Esse projeto foi desenvolvido como parte do curso DEVstart. A aplicação calcula a média das notas de atletas de ginástica artística, desconsiderando a maior e a menor nota.

## Como funciona

Em uma competição de ginástica artística, cada atleta recebe 5 notas de jurados diferentes. Porém a média final não considera a maior e a menor nota, apenas as 3 notas do meio.

O programa recebe o nome e as notas de cada atleta e mostra no console:
- Nome do atleta
- Notas obtidas (em ordem crescente)
- Média válida calculada

## Como executar

1. Ter o Node.js instalado no computador
2. Baixar o arquivo `notas-atletas.js`
3. Abrir o terminal na pasta do arquivo
4. Executar o comando:

```bash
node notas-atletas.js
```
